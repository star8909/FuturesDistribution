💰 비용 모델 (선물 vs 주식 차이)

주식(EquityDist): commission 1bp + slippage 2bps = 3~4bps

선물(이 프로젝트): commission 5bps + slippage 5bps + spread 2~3bps + roll 3~5bps/quarter = 15~50bps

백테스트 기본 50bps, stress test 100/150/200bps까지 검증.

🛡️ 실거래 안전 규칙 (한투 해외선물 적용 시)

1. Micro 선물만 사용 ($10k 계좌 권장: MES, MNQ, MGC, MCL)

2. 포지션 1개당 자본 5% 이하 (margin call 방지)

3. STOP 주문 항상 동시 입력 (한투 IF Done OCO 활용)

4. 추가증거금 발생 시 즉시 강제청산 (-15% drawdown 도달)

5. 롤오버 자동 알림 (FND 5영업일 전)

6. 24시간 모니터링 자동화 (직접 보지 X — 봇이 처리)
